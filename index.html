Project 01

Runway Detection System

1st Place — Honeywell Hackspace Hackathon
Python PyTorch Computer Vision U-Net OpenCV

Engineered a computer vision pipeline using a U-Net architecture to detect and analyze unstructured aerial imagery for aviation safety. The model was trained to perform pixel-level semantic segmentation on noisy, low-resolution satellite feeds — a domain where data scarcity and class imbalance posed significant preprocessing challenges.

The system achieved high accuracy on out-of-distribution imagery by combining custom data augmentation strategies with a hybrid loss function (Dice + Binary Cross-Entropy). Inference was optimized to run at near-real-time speed, making it viable for integration into aviation safety monitoring toolchains.

Project 02

Autonomous Chaos Engineering Platform

Python Kubernetes Docker Prometheus Isolation Forest ML Ensemble

Engineered a self-healing reliability platform managing a 5-node microservice architecture on Kubernetes. The platform autonomously injects fault scenarios (pod crashes, network partitions, resource exhaustion) and observes recovery behavior — forming a closed-loop chaos engineering system requiring zero manual intervention.

Developed a 3-model ML ensemble (Isolation Forest, Z-score analysis, CUSUM change-point detection) to autonomously detect system anomalies via live Prometheus telemetry streams. The ensemble votes on anomaly severity, triggers rollback procedures, and generates structured post-mortem reports with root cause attribution.

Project 03

SplitEase — Group Expense Platform

JavaScript PostgreSQL Node.js PWA Graph Algorithms REST API

Developed a full-stack Progressive Web App (PWA) to manage complex group finances, handling multi-currency transactions, recurring expenses, and multi-party settlements. The system was designed to feel native on mobile while remaining fully functional offline through service worker caching.

Architected a relational database using PostgreSQL with normalized schema design for expense categorization and audit trails. The core algorithmic innovation is a custom greedy debt-simplification algorithm — modeled as a directed weighted graph — that reduces the number of transactions required to settle a group from O(n²) to O(n), minimizing payment friction for all members.
