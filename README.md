## 🛠 Tech Stack

- **HTML5** — semantic structure
- **Tailwind CSS** (CDN) — utility-first styling
- **Vanilla JavaScript** — carousels, lightbox, scroll animations, active nav
- **Google Fonts** — Inter + JetBrains Mono
- **Material Symbols** — icons
- **Vercel** — hosting & deployment

---

## ✨ Features

- 🌙 Dark mode design with dot-grid background
- 📸 Image carousels with auto-advance, dot indicators, prev/next controls
- 🔍 Lightbox for full-size image viewing (click any project image)
- 🎞 Scroll-triggered fade-in reveal animations
- 🔗 Active nav link highlight as you scroll through sections
- 📱 Fully responsive — mobile, tablet, desktop
- ⚡ Zero build step — open `index.html` and it works

## 🚀 Running Locally

No build step needed. Just open the file:

```bash
# Clone the repo
git clone https://github.com/pushpitshukla28/portfolio.git
cd portfolio

# Open in browser
start index.html        # Windows
open index.html         # macOS
xdg-open index.html     # Linux
```

Or use VS Code's **Live Server** extension for hot reload.

## 🌐 Deployment

Deployed on **Vercel** via GitHub integration. Every push to `main` triggers an automatic redeploy.

```json
// vercel.json
{
  "cleanUrls": true,
  "trailingSlash": false
}
```
